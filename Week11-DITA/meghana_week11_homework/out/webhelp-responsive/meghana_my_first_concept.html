This provides an overview of the different type of issues that users often run into often in RSA NetWitness Platform version 11.x.

Users can often run into issues for licensing and dashboards within the RSA NetWitness Platform 11.x. License issues can be caused by the capture button within the System page of a service being disabled. Dashboard issues can be caused by multiple unintended "Default" dashboards.

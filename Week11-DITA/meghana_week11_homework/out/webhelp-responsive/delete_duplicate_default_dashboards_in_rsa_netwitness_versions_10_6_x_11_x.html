Delete duplicate "Default" dashboards in RSA NetWitness versions 10.6.x & 11.x

There are duplicate "Default" dashboards in the RSA NetWitness UI.

This task provides you with steps to remove any duplicate "Default" dashboards in the Dashboard section of the RSA NetWitness Platform UI.

  1. Log into the RSA NetWitness UI.
  2. Navigate to the Dashboards section of the UI.
    • In 10.6.x: the "Dashboard" section is located in the top left corner of the UI.
    • In 11.x: the "Dashboard" section is located in Monitor > Overview.
  3. Go into a "Default" dashboard.
  4. Next to the words "Default Dashboard," there are icons regarding dashboard options.

    In 10.6.x:

    • In between the "-" icon and the "Copy dashboard" icon, there is an icon with a red pencil.
    • Click the red pencil to show the dashboard editing options.
    • Click the option, "Restore Default Dashboard," which should clear all unnecessary/duplicate default dashboards.

    In 11.x:

    • In between the "-" icon and the "Copy dashboard" icon, there is an icon with a red pencil.
    • Click on that clockwise red arrow to "Restore Default Dashboards," which should clear all unnecessary/duplicate default dashboards.

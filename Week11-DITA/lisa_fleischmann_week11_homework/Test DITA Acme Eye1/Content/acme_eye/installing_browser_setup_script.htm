Installing Browser Setup Script and Logging in

Acme Eye is a proposal automation tool that enables you to save time and increase sales. To use Acme Eye, you must install the browser setup script.

For Acme Eye, you will need a desktop computer or mobile device, Microsoft Word, and a web browser. For additional information on reviewing prerequisites, see <xref href="reviewing_prerequisites.dita"/>.
Before logging in to Acme Eye for the first time, you need to install Acme Eye Browser Setup Script.

To install Acme Eye Browser Setup Script, follow these tasks.

  1. Download Acme Eye Browser Setup Script from the Acme webpage:<xref href="https://sales.acme.com" scope="external" format="html">Acme</xref>.
  2. Close all applications, including Outlook, and web browsers before installing Acme Eye Browser Setup Script.
  3. Double-click on Acme Eye Browser Setup Script executable file (.exe) to run the script.
  4. Log in to Acme Eye with your single sign-on credentials from the Acme Eye link: <xref href="https://eye.acme.com" scope="external" format="html">Acme Eye</xref>
When you log in to Acme Eye, the Welcome Screen is displayed.

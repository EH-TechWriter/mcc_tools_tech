Creating a Project

The first step to creating a document in Acme Eye is to create a new project.

To create a project, follow these steps:

  1. From the Welcome Screen use the Create New Project option, as shown in the Figure 3.1.
  2. Under Create New Project, enter the project information, as follows.
  3. Name: Name of the project.
  4. Type: Select the type of document you wish to create.
  5. Date: Date you plan on sending the document to your customer.
  6. Click CREATE.
Acme Eye creates the project and opens it for you. The name of your project appears followed by the type of document you are creating and the document outline.
